# delivery-system
Application de gestion de livraisons en Python
# 🚚 Système de livraison

## 📖 Description

**Système de livraison** est une application de gestion des livraisons développée en **Python** en utilisant les principes de la **Programmation Orientée Objet (POO)**.

L'application permet de gérer les différents acteurs d'un système de livraison, notamment les clients, les commandes et les livreurs, tout en calculant automatiquement les coûts de livraison.

---

## 🚀 Fonctionnalités

* 👤 Gestion des clients
* 📦 Gestion des commandes
* 🚚 Gestion des livreurs
* 💰 Calcul automatique du coût de livraison
* 🖥️ Interface graphique utilisateur
* 📊 Organisation du projet selon une architecture orientée objet

---

## 🛠️ Installation

### 1. Cloner le projet

```bash
git clone https://github.com/JoyceLumi/delivery_system.git
```

### 2. Accéder au dossier du projet

```bash
cd delivery_system
```

### 3. Installer les dépendances

```bash
pip install -r requirements.txt
```

---

## ▶️ Utilisation

Pour lancer l'application :

```bash
python main.py
```
